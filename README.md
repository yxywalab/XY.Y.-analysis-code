# Single-Cell RNA Sequencing Analysis Code

## Code Overview
This document provides the basic analysis code for our single-cell RNA sequencing study.

## ⚠️ Important Note

### Parameter Configuration
**Some parameters in the code require adjustment based on specific datasets**:

- **Variable Parameters**: Certain parameters are not fixed values
- **Adjustment Criteria**: Parameter modifications are based on:
  - Recommended parameters from original literature
  - Quality assessment of data integration
  - Our professional judgment on dataset characteristics
  - Quality control metrics results
